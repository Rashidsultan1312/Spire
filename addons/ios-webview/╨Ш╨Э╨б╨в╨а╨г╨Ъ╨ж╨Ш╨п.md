
## Установка

1. Скопировать `addons/ios-webview/` в проект
2. После экспорта iOS проекта запустить inject:

```bash
bash addons/ios-webview/ios/inject.sh build/ios/Game.xcodeproj
```

## Использование

### Открыть WebView

```gdscript
# Простой вызов — полноэкранный
Webview.open("https://example.com")

# С параметрами
Webview.open("https://example.com", {
    "fullscreen": false,
    "size": Vector2(0.9, 0.7),       # 90% ширины, 70% высоты экрана
    "position": "center",             # center / top / bottom
    "corner_radius": 16,
    "overlay_alpha": 0.5,             # затемнение фона
    "show_close_btn": true,
    "close_delay": 3,                 # показать кнопку ✕ через 3 сек
    "auto_dismiss": 30,               # закрыть через 30 сек
    "show_loading": true,             # спиннер пока грузится
    "transparent_bg": false,
    "bg_color": Color.BLACK,
    "bounce": true,                   # bounce при скролле
    "zoom": false,                    # пинч-зум
    "user_agent": "",                 # кастомный UA
    "js_on_load": "console.log('hi')", # JS после загрузки
    "custom_headers": {"X-App": "1"}, # доп. заголовки
    "clear_cache": false,
    "media_playback_requires_user_action": true,
})
```

### Управление

```gdscript
Webview.close()                         # закрыть
Webview.load_url("https://other.com")   # перейти на другой URL
Webview.go_back()                       # назад
Webview.go_forward()                    # вперёд
Webview.reload()                        # перезагрузить
Webview.evaluate_js("document.title")   # выполнить JS
Webview.is_open()                       # открыт ли
```

### Сигналы

```gdscript
Webview.opened.connect(_on_opened)
Webview.closed.connect(_on_closed)
Webview.page_loaded.connect(_on_loaded)       # func _on_loaded(url: String)
Webview.url_changed.connect(_on_url_changed)  # func _on_url_changed(url: String)
Webview.error.connect(_on_error)              # func _on_error(msg: String)
Webview.message_received.connect(_on_msg)     # func _on_msg(data: Variant)
```

### JS → GDScript мост

Из JS в вебвью можно отправить данные в GDScript:

```javascript
// В загруженной странице
window.webkit.messageHandlers.godot.postMessage({score: 100, name: "test"});
```

Поймать в GDScript:
```gdscript
Webview.message_received.connect(func(data): print("от JS: ", data))
```
